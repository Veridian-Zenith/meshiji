## Build and Run Instructions

### Debug Mode
To build and run in debug mode:
```bash
flutter run -d linux
```

### Release Mode
To build and run in release mode:
```bash
flutter build linux --release
```
### Run the Application
```bash
./build/linux/x64/release/bundle/vz_meshiji
```
